pluginFactory(clsToWrap, pluginName=None, commandArgParser=None)

source code 
The plugin factory takes a python class and wraps it into a maya MPxCommand. If no command name is passed then the name of passed class is used. The optional commandArgParser is a function for overriding the default handling of the maya argument data and should accept one arg, the command's MArgParser (or is it MArgDatabase? can't remember.)
