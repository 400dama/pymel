The background color of the window. The arguments correspond to the red, green, and blue color components. Each component ranges in value from 0.0 to 1.0. (Windows only flag)

Return the name of the front window. Note: you must supply the name of any window (the window does not need to exist). Returns unknownif the front window cannot be determined. In query mode, this flag needs a value.Flag can appear in Create mode of command Flag can appear in Edit mode of command Flag can appear in Query mode of command Flag can have multiple arguments, passed either as a tuple or a list.
