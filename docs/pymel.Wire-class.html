getCrossingEffect(self, **kwargs)

 
Set the amount of convolution effect. Varies from fully convolved at 0 to a simple additive effect at 1 (which is what you get with the filter off). Default is 0. This filter should make its way into all blend nodes that deal with combining effects from multiple sources. In query mode, this flag needs a value.
