command (Python) MEL version

nurbsCube

Go to: Synopsis. Flags. Return value. Related. Python examples.

Synopsis

nurbsCube([axis=[float, float, float]], [axisX=linear], [axisY=linear], [axisZ=linear], [constructionHistory=boolean], [curveOnSurface=boolean], [degree=int], [heightRatio=float], [lengthRatio=float], [name=string], [object=boolean], [patchesU=int], [patchesV=int], [pivot=[float, float, float]], [pivotX=linear], [pivotY=linear], [pivotZ=linear], [polygon=int], [range=boolean], [rebuild=boolean], [replaceOriginal=boolean], [width=linear])

Note: Strings representing object names and arguments must be separated by commas. This is not depicted in the synopsis.

nurbsCube is undoable, queryable, and editable.

The nurbsCube command creates a new NURBS Cube made up of six planes. It creates an unit cube with center at origin by default.
