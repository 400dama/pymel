getAllowOptionBoxes(self, **kwargs)

 
Indicate whether the menu will be able to support option box menu items. An error results if an option box item is added to a menu that doesn't allow them. This flag may be queried and must be specified when the popup menu is created. The default value is false. In query mode, this flag needs a value.
